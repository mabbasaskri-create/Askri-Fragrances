# Askri Fragrances

A premium luxury perfume eCommerce website built with **pure HTML, CSS, and Vanilla JavaScript** — no frameworks, no build step.

## Features
- Fully responsive (mobile, tablet, desktop)
- Black · White · Gold luxury theme
- 40 fake perfume products with images, ratings, reviews
- Working cart & wishlist (LocalStorage)
- Product search, filters, sorting, pagination
- Quick View modal, image gallery with zoom, tabs
- Testimonials slider, animated counters, FAQ accordion
- Newsletter popup, cookie banner, scroll-to-top
- SEO: meta tags, Open Graph, favicon, robots.txt, sitemap.xml

## Structure
```
Askri-Fragrances/
├─ index.html
├─ shop.html
├─ product.html
├─ about.html
├─ contact.html
├─ cart.html
├─ wishlist.html
├─ robots.txt
├─ sitemap.xml
├─ css/  (style.css, responsive.css)
├─ js/   (products.js, cart.js, script.js)
└─ images/ (favicon.svg — product photos load from Unsplash)
```

## Run
Just open `index.html` in a browser. That's it.
